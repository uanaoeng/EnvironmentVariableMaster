软件说明
========
系统自带的环境变量管理器在编辑环境变量时，变量值和变量名都显示在一行当中，很不方便（特别是编辑path变量时），所以写了这个工具。

使用这款软件添加或编辑变量时，为方便查看变量的内容，软件自动在每个以分号";"相隔的目录后面添加了换行符，保存时，自动删除变量中所有的换行符。
